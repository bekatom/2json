# Test SQL databases

Here you can run test for compatibility with multiple SQL different samples from different sources.

To run all tests in Node.js: 

```js
    mocha .
```

To run specific test:
```js
    mocha worldwind.js
    mocha world.js

```
All these databases belong to their owners.
